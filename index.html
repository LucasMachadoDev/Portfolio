Conhecimentos

HTML5

Com as mudanças do HTML5 otimizar sites e sistemas de buscas do Google ficou ainda mais fácil. Disponível em diversas plataformas e com ótima performance em computadores, dispositivos móveis e tablets.

Pré-processadores CSS

Os pré-processadores facilitam a manutenção do código, podendo também serem organizados através de uma importação quando for necessário uma maior flexibilidade e reutilização

JavaScript

Alta compatibilidade com plataformas, sistemas e navegadores de web. Mais leve e rápida do que outras linguagens de programação, tornando as páginas de internet mais dinâmicas e interativas, características essenciais do UX. Com navegadores que interpretam a linguagem por conta própria, tirando a necessidade de usar um compilador.

Git e GitHub

Confiabilidade: com um sistema centralizado evita que os problemas surjam no servidor impedindo o desenvolvimento de todo um trabalho. Um sistema distribuído impede riscos, pois os repositórios dos programadores servem como cópias de backup permitindo que o trabalho de um time não seja interrompido por problemas de servidor.


Redução de custos com o servidor: o repositório central funciona como repositório oficial, ao invés de ter uma função de processador central dos pedidos. Assim, a carga de processamento é distribuída pelas máquinas dos programadores.

PHP

Atualmente estudando

É Multiplataforma. É a facilidade que os usuários têm de poder usar e rodar a linguagem PHP numa variedade de sistemas operacionais. Windows, Linux (e suas distribuições) e MacOS são alguns deles. O mesmo acontece entre os navegadores: Chrome, Safari, Edge, Firefox e Opera, entre outros, são todos compatíveis.


Alto Desempenho. O PHP é capaz de suportar grandes quantidades de dados. Com isso, a linguagem consegue executar muitas funções e consumir muitos recursos ao mesmo tempo. E sem comprometer o desempenho e a velocidade do servidor em que está hospedado.

Design

Super flexibilidade: sites com web design responsivo são fluídos, isto é, seus conteúdos se adaptam livremente a todas as resoluções de tela e todos os dispositivos, tanto os grids quanto as imagens. Assim como um líquido se espalha ou dilui para permitir que o conteúdo caiba num espaço apertado mantendo a aparência, a fluidez do web design responsivo alcança o mesmo resultado com o conteúdo do website em uma tela.


Baixo custo: As vantagens em ter um único site que se adéqua a qualquer tipo de uso em todos os dispositivos são significativas, quando comparadas a ter dois sites separados e a economia é substancial. Sites planejados apenas para celulares não oferecem as vantagens de navegação que um site desenhado de forma tradicional para desktops. O web design responsivo melhora a otimização do site para ferramentas de busca (SEO), uma vez que tem todos os visitantes direcionados para um único site, não importa qual dispositivo eles preferem usar.
